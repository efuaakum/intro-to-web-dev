Required Tools:

  • VS Code Online (vscode.dev)
  • Git

System Requirements:

  • Operating System: Git is compatible with Windows (8.1 or later, 64-bit or 32-bit), macOS (11.0 or later), and various Linux distributions.
  • Processor: x86-based or similar architecture is generally required.
  • Storage/Disk Space: A few megabytes (around 20MB) of storage space is usually enough.
  • RAM: 4GB
  • Internet Access: Required for downloading the installer
  • Administrator Rights (Windows): Administrator privileges are needed to install Git on Windows.
